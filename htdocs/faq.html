Frequently Asked Questions - ASF Boxer

As this service is brand new, our FAQ list is brief. Please ask your question on `users@infra.apache.org` and we'll add questions and answers to this page.

If I change my LDAP password, do I have to reauthenticate with GitBox?
You do not need to reauthenticate.
If I unlink my Git account, can I still do commits using GitBox?
Yes, GitBox is meant for users wishing to commit to Git without having to worry about GitHub (some users have concerns about GitHub's terms of service). There is no requirement at the ASF that you use GitHub for anything. Projects may track issues and PRs on GitHub, but you can commit code solely through GitBox if you wish.
How do I add or remove a repository from the list of repos to which i have write access?
We authomatically manage your repository access using your ASF project access list. New repositories in a project you are a part of will be automatically included in your list of repositories within a few minutes of the repository being created. You cannot get removed from a single repository unless you leave its project and thus get removed from all the project's repositories.
